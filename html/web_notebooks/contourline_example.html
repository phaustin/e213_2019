make the plot with specified lines and labels¶

They don’t have to be the same – you can label only some of the lines. You can control line thickness, dashed/solid/dotted etc., see the matplotlib docs.

[3]:

fig, ax = plt.subplots(1, 1, figsize=(15, 10))
the_levels = [0.2, 0.6, 1.0, 1.4, 1.8, 2.0, 2.2, 2.6]
cs = ax.contour(x1, y1, psi, levels=the_levels, colors="k")
ax.set_aspect(1)
ax.clabel(cs, the_levels, fmt="%3.1f")
ax.grid(True)
